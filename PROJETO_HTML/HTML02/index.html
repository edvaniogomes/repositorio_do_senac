Subtítulo de Seção

HTML, que significa HyperText Markup Language , é a lingugaem usada para estruturar páginas na web. Você pode enfatizar o texto ou torna-lo fortecom comandos simples.

Lista de Ferramentas

  • Editor de texto
  • Navegador da web
  • Conhecimento básico de HTML

Passos para se construir uma página

  1. Abrir um editor de texto
  2. Escrever código HTML
  3. Salvar o arquivo com a extensão .html
  4. Abrir o arquivo no navegador
